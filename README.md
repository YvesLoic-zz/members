# SM
 Gestionnaire pour les espaces membres avec prise en charge des écoles, élèves et presentation des paiements.
# Procédure de test
 - installer les dépendances
~~~~
    composer install
~~~~
 - créer la bd mysql
~~~~
    php artisan create:db
~~~~
 - créer les tables en bd
~~~~
  php artisan migrate --seed
~~~~
 - démarrer le projet
~~~~
    php artisan serve
~~~~
